## <br>정렬(Sorting)

- 특정 값을 기준으로 데이터를 순서대로 배치하는 방법

- 구현 난이도는 쉽지만 속도는 느린 알고리즘
  - 버블 정렬, 삽입 정렬, 선택 정렬

- 구현 난이도는 조금 어렵지만 속도는 빠른 알고리즘
  - 합병 정렬, 힙 정렬, 퀵 정렬, 트리 정렬

- 하이브리드 정렬
  - 팀 정렬, 블록 병합 정렬, 인트로 정렬

- 기타 정렬 알고리즘
  - 기수 정렬, 카운팅 정렬,셸 정렬,보고 정렬

### 버블 정렬(Bubble sort)

- 인접한 데이터를 비교하며 자리 바꾸는 방식
- 알고리즘 복잡도: O(n<sup>2</sup>)
