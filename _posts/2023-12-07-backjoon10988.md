### 문제

https://www.acmicpc.net/problem/10988



### 문제 풀이

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.next();
        int answer = 1;

        for (int i = 0; i < s.length() / 2; i++) {
            if (s.charAt(i) != s.charAt(s.length() - (i + 1))) {
                answer = 0;
                break;
            }
        }

        System.out.println(answer);
    }
}
```
