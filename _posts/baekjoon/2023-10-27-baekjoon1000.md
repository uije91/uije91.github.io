### 문제

[https://www.acmicpc.net/problem/1000](https://www.acmicpc.net/problem/1000)

### <br>문제 풀이

```java
import java.util.*;
public class Main{
	public static void main(String args[]){
		Scanner sc = new Scanner(System.in);
		int a, b;
		a = sc.nextInt();
		b = sc.nextInt();
		System.out.println(a + b);
	}
}
```
