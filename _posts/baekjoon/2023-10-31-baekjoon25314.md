### 문제

https://www.acmicpc.net/problem/25314



### 문제 풀이

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        //바이트의 크기
        int n = sc.nextInt();

        for (int i = 0; i < n/4; i++) {
            System.out.print("long ");
        }
        System.out.print("int");
    }
}

```
