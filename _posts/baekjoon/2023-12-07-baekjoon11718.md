### 문제

https://www.acmicpc.net/problem/11718



### 문제 풀이

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        while (sc.hasNext()) {
            System.out.println(sc.nextLine());
        }
    }
}
```
