### 문제

https://www.acmicpc.net/problem/1330



### 문제 풀이

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        if(a>b) System.out.println(">");
        else if(a<b) System.out.println("<");
        else if(a==b) System.out.println("==");
    }
}
```
