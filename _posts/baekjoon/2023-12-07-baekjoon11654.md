### 문제

https://www.acmicpc.net/problem/11654



### 문제 풀이

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        char c = sc.next().charAt(0);
        System.out.println((int)c);
    }
}
```
