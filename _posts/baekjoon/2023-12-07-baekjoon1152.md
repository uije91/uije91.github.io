### 문제

https://www.acmicpc.net/problem/1152



### 문제 풀이

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine();
        StringTokenizer st = new StringTokenizer(s," ");
        System.out.println(st.countTokens());
    }
}
```
