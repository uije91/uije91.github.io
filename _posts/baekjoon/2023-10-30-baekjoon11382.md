### 문제

https://www.acmicpc.net/problem/11382



### 문제 풀이

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        //받는 숫자가 int형보다 큰 값을 출력하므로 long형 을 이용하여 출력
        Scanner sc = new Scanner(System.in);
        long a = sc.nextLong();
        long b = sc.nextLong();
        long c = sc.nextLong();

        System.out.println(a+b+c);
    }
}
```
