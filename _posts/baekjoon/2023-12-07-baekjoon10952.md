### 문제

https://www.acmicpc.net/problem/10951



### 문제 풀이

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A, B;

        while (sc.hasNext()) {
            A = sc.nextInt();
            B = sc.nextInt();
            System.out.println(A + B);
        }
    }
}
```
