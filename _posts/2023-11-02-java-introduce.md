## 자바(Java)란?



### 자바의 특징

- 플랫폼 독립적
  - 운영체제에 따라 코드를 재 작성하지 않아도 됩니다.
- 객체 지향적
  - 절차 지향적 언어(문제를 풀이하기 위해 순서대로 처리)에 비해 문제를 각각 처리하여 생산성 및 보안성이 높습니다.

### 자바 플랫폼(Java Platform)

- Java SE(Standard Edition)
  - 기본 개발 환경
- Java EE(Enterprise Edition)
  - 서버 기반 프로그램 개발 환경
- Java ME(Micro Edition)
  - 모바일, 임베디드 시스템 개발 환경
